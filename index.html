Grafiklere yeni doğru ekleme

R de çizdirilmiş bir grafiğe yeni bir doğru eklemek için

abline fonksiyonu kullanılır.

abline üç farklı şekilde kullanılabilir.

1. abline(a=...,b=...) kullanılırsa a+bx doğrusunu

grafiğe ekler

R
plot(cars,xlim=c(0,30))
abline(a=10,b=2)  #10+2*x doğrusu
abline(a=50,b=-3,lwd=2,lty=3,col="red") #50-3*x doğrusu
						# lwd doğru kalınlığı 
						# lty doğru tipi
						# col ise rengi

2. kullanım dikey doğru ekler abline(v=...)

3. kullanım yatay doğru ekler abline(h=...)

R
plot(x=iris$Sepal.Length,y=iris$Sepal.Width)
abline(v=6,lwd=2,col=3,lty=5)
abline(h=3.5,lwd=2,col=4,lty=4)

Yeni fonksiyon oluşturma

R'de hazır yüklü fonksiyonlar bizim için her durumda

yeterli olmayacaktır. Bazı durumlarda kendi fonksiyon-

larımızı yazmamız gerekecektir.

Genel fonksiyon tanımlama yöntemi:

fonk_adı <-function(argümanlar){

fonksiyon işlemleri

}

Ör: Bir sayının küp değerini hesaplayan bir fonksiyon

yazın

R
kup<-function(x){

x^3

}
R
kup(15)
kup(-3)
kup(1:10)  # Not: fonksiyon içindeki işlem vektörlerle veya
		# matrislere uyumlu ise fonksiyon da uyumlu 
		# olacaktır

Ör2: R'de var fonksiyonu bir vektör için örneklem

varyansı hesaplar. Örneklem varyansında paydada n-1

vardır. Kitle varyansında paydada n vardır.

R'de kitle varyansı hesaplayan bir fonksiyon oluşturun.

R
kitle_var<-function(x){

n<-length(x)

sum( ( x-mean(x) )^2 )/n

}
R
kitle_var(x=1:5)
var(1:5)  # Örneklem varyansı fonksiyonu R'de ön tanımlı 

n          # R'de fonksiyon içinde tanımlanan
		# değişkenler kaydedilmez. Dolayısıyla
		# böyle değerleri çağırdığımızda 
		# hata sonucu alırız.

Ör: Kitle standart sapması hesaplayan bir fonksiyon

oluşturun

R
kitle_sd<-function(x){

sqrt(kitle_var(x))

}
R
kitle_sd(1:5)  # Not: Kendi tanımladığımız eski bir
	# fonksiyonu yeni bir fonksiyon içinde kullanabiliriz.

sd(1:5)  # R'de ön tanımlı örneklem standart sapması fonksiyonu.

Ör: For döngüsü kullanarak eklemeli toplam hesaplayan

eklemeli isminde bir fonksiyon oluşturun.

Girdi: x (bir vektör)

Çıktı: bir vektör

R
eklemeli<-function(x){

n<-length(x)

sonuc<-0

for(i in 1:n){

sonuc[i]<-sum(x[1:i])

}

sonuc

}
R
x<-c(2,3,1,5,7,5,4)

eklemeli(x)

fibo<-function(n){

dizi<-rep(0,n)

dizi[1:2]<-1

for(i in 3:n){

dizi[i]<-dizi[i-1]+dizi[i-2]

}

dizi

}
